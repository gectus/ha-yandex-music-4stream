# Yandex Music for 4STREAM

Кастомная интеграция Home Assistant для стриминга музыки из Яндекс.Музыки на аудиоустройства 4STREAM (Arylic).

## Возможности

- Воспроизведение музыки из Яндекс.Музыки на устройствах 4STREAM/Arylic
- Просмотр и выбор треков через панель «Мультимедиа» Home Assistant
- Просмотр избранных треков («Мне нравится») и пользовательских плейлистов
- Очередь воспроизведения с поддержкой перемешивания и повтора
- Автоматический переход к следующему треку
- Управление громкостью, паузой, перемоткой
- Поиск треков по текстовому запросу
- Поддержка нескольких устройств 4STREAM одновременно
- Встроенный HTTP-прокси для совместимости с устройствами (HTTPS → HTTP)

## Требования

- Home Assistant 2024.1 или новее
- Подписка Яндекс.Плюс (без неё доступны только 30-секундные превью)
- Одно или несколько устройств 4STREAM/Arylic в локальной сети

## Установка

### Ручная установка

1. Скопируйте папку `custom_components/yandex_music_4stream` в директорию `custom_components` вашего Home Assistant.
2. Перезапустите Home Assistant.

### HACS (в будущем)

Интеграция будет доступна через HACS после публикации.

## Настройка

### Шаг 1: Получение OAuth-токена Яндекс.Музыки

1. Откройте в браузере:
   ```
   https://oauth.yandex.ru/authorize?response_type=token&client_id=23cabbbdc6cd418abb4b39c32c41195d
   ```
2. Войдите в свой аккаунт Яндекса (если ещё не вошли).
3. Нажмите «Разрешить».
4. Скопируйте токен из URL — это значение параметра `access_token`.

### Шаг 2: Добавление интеграции в Home Assistant

1. Перейдите в **Настройки → Устройства и службы → Добавить интеграцию**.
2. Найдите «Yandex Music for 4STREAM».
3. Введите OAuth-токен из шага 1.
4. Введите IP-адрес вашего устройства 4STREAM.
5. При необходимости добавьте дополнительные устройства.

После настройки для каждого устройства будет создана сущность `media_player`.

## Использование

### Панель «Мультимедиа»

1. Откройте боковое меню → **Мультимедиа**.
2. Выберите **Yandex Music**.
3. Просматривайте «Мне нравится» или «Мои плейлисты».
4. Выберите устройство для воспроизведения в нижней части экрана.
5. Нажмите на трек — музыка начнёт играть.

### Карточки дашборда

Добавьте на дашборд карточку `media-control`:

```yaml
type: media-control
entity: media_player.your_device_name
```

Карточка покажет текущий трек, обложку, кнопки управления.

### Сервисы

Интеграция поддерживает стандартные сервисы `media_player`:

```yaml
# Воспроизвести плейлист «Мне нравится»
service: media_player.play_media
target:
  entity_id: media_player.your_device
data:
  media_content_type: music
  media_content_id: liked

# Воспроизвести конкретный плейлист
service: media_player.play_media
target:
  entity_id: media_player.your_device
data:
  media_content_type: music
  media_content_id: "playlist:1234"

# Поиск и воспроизведение
service: media_player.play_media
target:
  entity_id: media_player.your_device
data:
  media_content_type: music
  media_content_id: "Radiohead Creep"
```

### Ключевые решения

**HTTP-прокси (StreamProxy)**

Устройства 4STREAM не умеют воспроизводить HTTPS-ссылки. Интеграция включает встроенный HTTP-прокси, который:
- Принимает HTTP-запрос от устройства
- Загружает аудио по HTTPS с серверов Яндекса
- Стримит данные чанками (64 КБ) без полной буферизации
- Поддерживает Range-запросы для перемотки

**Just-in-time получение URL**

Прямые ссылки на треки живут ~1 минуту. Поэтому интеграция хранит очередь из `track_id`, а URL запрашивается непосредственно перед воспроизведением каждого трека.

## Устранение неполадок

### Интеграция не загружается после перезапуска HA

Если при старте HA сеть ещё не готова, может возникнуть DNS-ошибка при подключении к Яндексу. Интеграция использует `ConfigEntryNotReady` — HA автоматически повторит попытку через несколько минут.

### Музыка не играет

1. Проверьте, что устройство 4STREAM доступно по сети (откройте `http://<ip>/httpapi.asp?command=getStatusEx` в браузере).
2. Проверьте, что у вас активна подписка Яндекс.Плюс.
3. Посмотрите логи: **Настройки → Система → Журналы**, фильтр `yandex_music_4stream`.

### Обложки не отображаются

Обложки загружаются с серверов Яндекса по HTTPS. Если HA находится за reverse proxy, убедитесь, что он корректно проксирует запросы.
